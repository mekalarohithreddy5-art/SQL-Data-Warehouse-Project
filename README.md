# Data Warehouse and Analytics Project

## Overview
This project demonstrates the design and implementation of a modern Data Warehouse using Microsoft SQL Server. It covers the complete workflow from importing raw CSV data to building a structured data warehouse and generating business insights through SQL analysis.

The project follows industry-standard data engineering practices and is intended to strengthen practical SQL and Data Engineering skills.

---

## Project Objectives

### Data Engineering
- Import data from multiple CSV files.
- Clean and validate the data.
- Design a structured data warehouse.
- Create dimension and fact tables.
- Build an optimized data model for analytics.

### Data Analytics
Perform SQL analysis to answer business questions such as:
- Customer behavior analysis
- Product performance analysis
- Sales trend analysis
- Revenue and business insights
